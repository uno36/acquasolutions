# Acquasolutions <a name="about-project"></a>

**Acquasolutions** specializes in water treatment and borehole installations, offering expert solutions for clean water access. With a focus on sustainability and quality, AquaSolutions provides tailored services to meet diverse needs, ensuring safe and reliable water sources for communities, industries, and households alike.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 npm run dev
```

### Setup

Clone this repository to your desired folder:

```sh  
  git clone git@github.com:uno36/acquasolutions.git
  cd acquasolutions  
```

### Install

Install this project with:

```sh
  cd acquasolutions
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm run dev
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```
